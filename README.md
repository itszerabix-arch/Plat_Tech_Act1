# Plat_Tech_Act1

JAVASCRIPT FUNCTIONS & EVENTS DEMO
===================================

This application illustrates JavaScript functions and event handling via an engaging interface. It highlights different kinds of functions and event listeners in a practical and accessible manner.

FILES INCLUDED:
1. index.html - Main HTML structure with Bootstrap integration
2. style.css - Custom CSS styles and overrides
3. script.js - All JavaScript functions and event listeners
4. README.txt - This file

HOW TO USE:
1. Open index.html in any modern web browser
2. Enter your name in the input field
3. Try clicking all the buttons to see functions in action
4. Hover over the "Hover Over Me" button
5. Double-click the "Double Click Me" button
6. Type in the keyboard input field to see key events
7. Watch the event log update in real-time
8. Use the "Reset All" button to restore initial state

NOTES:
- This is a front-end only application, no server required
- All code is contained within these three files
- No external dependencies besides Bootstrap and Font Awesome CDN
- Fully responsive design that works on mobile and desktop
- Code is commented for educational purposes
